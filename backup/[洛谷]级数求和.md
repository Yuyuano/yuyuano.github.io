# P1035 [NOIP 2002 普及组] 级数求和
## 题目
现已知Sn=1+1/2+1/3+1/n 
Sn>k 现给出一个整数 k，要求计算出一个最小的 n，使得 Sn>k
### 代码案例
```python
k = int(input()) #K的值
n = 0 #N的初始值
S = 0.0  #S的初始值
while S <= k: #计算循环
    n += 1
    S += 1 / n
print(n)
```
依照题目要求，**k**为输入的值，**n**与**S**为还没计算的初始值
