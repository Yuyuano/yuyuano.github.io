# 利用简单的IF语句来进行年份判断
## 条件
>[!NOTE]
闰年的条件为4年一润，百年不润，四百年再润
>
也就是说：

* 能被 4 整除但不能被 100 整除的是闰年；
* 或者能被 400 整除的也是闰年。
  
*即*
```python
nian=int(input()) #设置变量年
if nian % 4 == 0 and nian % 100 != 0 or nian % 400 == 0:#判断闰年条件
    print('Yes')
else:
    print('No')
```
这算是补之前的作业内容

:spoiler[因为我觉得太简单了不需要]
>[!TIP]
在判断条件中，我们使用了 or（或）来连接两个关键判断：
* 年份能被 4 整除但不能被 100 整除；
* 或者能被 400 整除。
这两个条件是并列的，只要满足其中之一，就可以判断为闰年。
