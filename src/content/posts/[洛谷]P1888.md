# P1888 三角函数

## 题目描述

输入一组勾股数 $a,b,c（a\neq b\neq c）$，用分数格式输出其较小锐角的正弦值。（要求约分。）

## 输入格式

一行，包含三个正整数，即勾股数 $a,b,c$（无大小顺序）。

## 输出格式

一行，包含一个分数，即较小锐角的正弦值

## 输入输出样例 #1

### 输入 #1

```
3 5 4
```

### 输出 #1

```
3/5
```

## 说明/提示

数据保证：$a,b,c$ 为正整数且 $\in [1,10^9]$。

简而言之，就是给你三个边的参数，要你自己找斜边，并计算最小锐角的正弦值
以下是代码：
```python
import math
a,b,c=map(int,input().split())
xie_bian=max(a,b,c)#取最大的一个数
if a == xie_bian:#做斜边判断
    leg1,leg2=b,c
elif b == xie_bian:
    leg1,leg2=a,c
else:
    leg1,leg2=a,b
duan_bian=min(leg1,leg2)
g=math.gcd(xie_bian,duan_bian)
print(f"{xie_bian//g}/{duan_bian//g}")
```
>[!NOTE]
为什么要引用math库呢，因为要用到math中的gcd函数变量来对分数进行约分
>
#### gcd函数原理讲解
gcd可以用if语句来进行表述，如下：
```python
def gcd(x, y):
    while y:
        x, y = y, x % y
    return x
```
