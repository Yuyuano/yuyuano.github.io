# P1888 三角函数

## 题目描述

输入一组勾股数 $a,b,c（a\neq b\neq c）$，用分数格式输出其较小锐角的正弦值。（要求约分。）

## 输入格式

一行，包含三个正整数，即勾股数 $a,b,c$（无大小顺序）。

## 输出格式

一行，包含一个分数，即较小锐角的正弦值

## 输入输出样例 #1

### 输入 #1

```
3 5 4
```

### 输出 #1

```
3/5
```

## 说明/提示

数据保证：$a,b,c$ 为正整数且 $\in [1,10^9]$。

简而言之，就是给你三个边的参数，要你自己找斜边，并计算最小锐角的正弦值，最小锐角一定是短直角边所对的角，因此 sin 值 = 短直角边 / 斜边。
以下是代码：
```python
import math

# 读入三个直角边
a, b, c = map(int, input().split())

# 1. 斜边一定是最大的那条
hypotenuse = max(a, b, c)

# 2. 剩下两条就是直角边，取其中较小的那条
if a != hypotenuse and b != hypotenuse:
    shorter_leg = min(a, b)
elif a != hypotenuse and c != hypotenuse:
    shorter_leg = min(a, c)
else:  # b、c 是直角边
    shorter_leg = min(b, c)

# 3. 约分后输出：短直角边 / 斜边
g = math.gcd(shorter_leg, hypotenuse)
print(f"{shorter_leg // g}/{hypotenuse // g}")
```
>[!TIP]
为什么要引用math库呢，因为用 math.gcd 求最大公约数，把分子分母同时除掉它即可得到最简分数。
>
#### gcd函数原理讲解
gcd可以用if语句来进行表述，如下：
```python
def gcd(x, y):
    while y:
        x, y = y, x % y
    return x
```
