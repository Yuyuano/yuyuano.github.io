# Python 运算符表

## 1. 算术运算符

| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | `8` |
| `-` | 减法 | `5 - 3` | `2` |
| `*` | 乘法 | `5 * 3` | `15` |
| `/` | 除法（返回浮点数） | `5 / 3` | `1.666...` |
| `//` | 整除（向下取整） | `5 // 3` | `1` |
| `%` | 取模（余数） | `5 % 3` | `2` |
| `**` | 幂运算 | `5** 3` | `125` |


## 2. 赋值运算符

| 运算符 | 描述 | 示例 | 等价于 |
|--------|------|------|--------|
| `=` | 基本赋值 | `x = 5` | - |
| `+=` | 加法赋值 | `x += 3` | `x = x + 3` |
| `-=` | 减法赋值 | `x -= 3` | `x = x - 3` |
| `*=` | 乘法赋值 | `x *= 3` | `x = x * 3` |
| `/=` | 除法赋值 | `x /= 3` | `x = x / 3` |
| `//=` | 整除赋值 | `x //= 3` | `x = x // 3` |
| `%=` | 取模赋值 | `x %= 3` | `x = x % 3` |
| `**=` | 幂运算赋值 | `x**= 3` | `x = x **3` |
| `&=` | 按位与赋值 | `x &= 3` | `x = x & 3` |
| `\|=` | 按位或赋值 | `x \|= 3` | `x = x \| 3` |
| `^=` | 按位异或赋值 | `x ^= 3` | `x = x ^ 3` |
| `>>=` | 右移赋值 | `x >>= 3` | `x = x >> 3` |
| `<<=` | 左移赋值 | `x <<= 3` | `x = x << 3` |


## 3. 比较运算符

| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 3` | `False` |
| `!=` | 不等于 | `5 != 3` | `True` |
| `>` | 大于 | `5 > 3` | `True` |
| `<` | 小于 | `5 < 3` | `False` |
| `>=` | 大于等于 | `5 >= 3` | `True` |
| `<=` | 小于等于 | `5 <= 3` | `False` |


## 4. 逻辑运算符

| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `and` | 逻辑与 | `True and False` | `False` |
| `or` | 逻辑或 | `True or False` | `True` |
| `not` | 逻辑非 | `not True` | `False` |


## 5. 位运算符

| 运算符 | 描述 | 示例（x=5即101, y=3即011） | 结果 |
|--------|------|--------------------------|------|
| `&` | 按位与 | `x & y` | `1`（001） |
| `\| ` | 按位或 | `x \| y` | `7`（111） |
| `^` | 按位异或 | `x ^ y` | `6`（110） |
| `~` | 按位取反 | `~x` | `-6`（补码） |
| `<<` | 左移 | `x << 1` | `10`（1010） |
| `>>` | 右移 | `x >> 1` | `2`（10） |


## 6. 成员运算符

| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `in` | 存在于序列中 | `3 in [1,2,3]` | `True` |
| `not in` | 不存在于序列中 | `3 not in [1,2,3]` | `False` |


## 7. 身份运算符

| 运算符 | 描述 | 示例 | 结果 |
|--------|------|------|------|
| `is` | 两个对象是否相同（内存地址） | `x is y` | `True/False` |
| `is not` | 两个对象是否不同（内存地址） | `x is not y` | `True/False` |


## 8. 运算符优先级（从高到低）

1. `**`（幂运算）
2. `~`（按位取反）
3. `*` `/` `//` `%`（乘除、整除、取模）
4. `+` `-`（加减）
5. `<<` `>>`（位移）
6. `&`（按位与）
7. `^` `|`（按位异或、或）
8. `<=` `<` `>` `>=`（比较）
9. `==` `!=`（等于、不等于）
10. `=` `%=` `/=` `//=` `-=` `+=` `*=` `**=`（赋值）
11. `is` `is not`（身份判断）
12. `in` `not in`（成员判断）
13. `not` `and` `or`（逻辑运算）
