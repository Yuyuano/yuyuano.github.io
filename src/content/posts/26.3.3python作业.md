# 23.3.3Python作业
## 第一题
#### 题目：
输入`a`、`b`、`c` 3个参数，求解`ax²+bx+c=0`的两个根，假定`b²-4ac>0`
#### 讲解：
其本质就是求解一元二次方程的根，根据公式：
$$
x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}
$$
就可以推导出两个根为：
$$
x_1=\frac{-b+\sqrt{b^2-4ac}}{2a}
$$
$$
x_2=\frac{-b-\sqrt{b^2-4ac}}{2a}
$$

Python代码实现如下：

```python
import math

a, b, c = map(int, input().split())
delta = b**2 - 4*a*c#计算判别式
x1 = (-b + math.sqrt(delta)) / (2*a)#计算第一个根
x2 = (-b - math.sqrt(delta)) / (2*a)#计算第二个根

print(x1, x2)
```
## 第二题
#### 题目：
输入`a`、`b`、`c` 3个参数，以他们作为三角形的三条边，判断是否可以构成一个三角形，如能则进一步计算其面积。三角形的面积S可以通过以下公式计算：


$$
S=\sqrt{p(p-a)(p-b)(p-c)}
$$

其中`P=(a+b+c)/2`

#### 讲解：
首先需要判断输入的三条边是否可以构成一个三角形，条件是任意两边之和大于第三边，即：
$$
a+b>c
$$
$$
b+c>a
$$
$$
c+a>b
$$
如果满足以上条件，则可以构成一个三角形。接下来计算三角形的面积，首先计算半周长`P`，然后使用公式计算面积。
Python代码实现如下：

```python
import math
a, b, c = map(int, input().split())
if a+b>c and b+c>a and c+a>b: #判断是否构成三角形
    p = (a+b+c)/2 #计算半周长
    s = math.sqrt(p*(p-a)*(p-b)*(p-c)) #计算三角形面积
    print(s)
else:
    print("不能构成三角形")
```

#### 题目：
输入一个字母，判断它是否为小写字母

#### 讲解：
判断一个字母是否为小写字母，只需要判断它的ASCII码是否在`97`到`122`之间即可。
Python代码实现如下：

```python
letter = input()
if 97 <= ord(letter) <= 122: #ord()函数可以返回字符的ASCII码
    print("是小写字母")
else:
    print("不是小写字母")
```
