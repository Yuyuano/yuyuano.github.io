# 26.3.9Python作业

## 第一题
#### 题目：用户输入一个数字，判断是否为质数
#### 讲解：质数是指只能被1和它本身整除的自然数。判断一个数是否为质数，可以通过以下步骤：
1. 如果输入的数小于2，则不是质数。
2. 如果输入的数等于2，则是质数。
3. 如果输入的数大于2，则需要判断它是否能被2到√n之间的任何数整除。如果不能，则是质数；如果能，则不是质数。
Python代码实现如下：

```python
n = int(input())
if n < 2:
    print('No')
else:
    for i in range(2, int(n**0.5) + 1):
        if n % i == 0:  # 能整除，说明不是质数
            print('No')
            break
    else:  # 循环正常结束（没有被break），说明是质数
        print('Yes')
```
## 第二题
#### 题目：打印九九乘法表
Python代码实现如下：

```python
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f'{j}*{i}={i*j}', end=' ')
    print()  # 换行
```

## 第三题
#### 题目：公鸡5元一只，母鸡3元一只，小鸡1元三只，用100元买100只鸡，问公鸡、母鸡、小鸡各多少只？
#### 讲解：这是一个经典的数学问题，可以通过枚举法来解决。我们可以设公鸡的数量为x，母鸡的数量为y，小鸡的数量为z，根据题意可以得到以下两个方程：
$$
x+y+z=100
$$
$$
5x+3y+z/3=100
$$
我们可以通过枚举x、y、z的所有可能值，来求解这个问题。Python代码实现如下：
```python
for x in range(0, 21):
    for y in range(0, 34):
        z = 100 - x - y
        if 5*x + 3*y + z/3 == 100:
            print(f'公鸡{x}只，母鸡{y}只，小鸡{z}只')
```

## 第五题
#### 题目: 计算1-1/3+1/5-1/7+...-1/97+1/99的和
#### 讲解：这是一个交替级数，可以通过循环来计算。我们可以使用一个变量来累加结果，并根据当前项的奇偶性来决定是加还是减。

Python代码实现如下：

```python
total = 1
count = 0
for i in range(3, 100, 2):
    count += 1
    if count % 2 != 0:
        total -= 1/i
    else:
        total += 1/i
print(total)
```
