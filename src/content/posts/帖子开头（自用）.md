# 给自己自用的帖子开头备注
```txt
---
title: 帖子开头frontmatter
published: 2025-11-07
pinned: false
description: while循环版
tags: [作业]
category: Python
licenseName: 转载时转载请注明出处
author: 遇缘
draft: false
date: 2025-11-07
pubDate: 2025-11-07
---
```
| 字段            | 值                                             | 含义                                                   |
| ------------- | --------------------------------------------- | ---------------------------------------------------- |
| `title`       | **Markdown Tutorial**                         | 文章标题，会出现在浏览器标签、列表页、SEO 等位置。                          |
| `published`   | **2025-01-20**                                | 正式“发表”日期，可用于排序、归档、RSS。                               |
| `pinned`      | **true**                                      | 置顶标志，主题或列表页可把这篇文章固定到最前面。                             |
| `description` | **A simple example of a Markdown blog post.** | 摘要/副标题，用于 SEO、社交媒体卡片、列表页缩略。                          |
| `tags`        | **\[Markdown, Blogging]**                     | 标签，方便按关键词聚合、搜索、做标签云。                                 |
| `category`    | **Examples**                                  | 大分类，可做顶部导航或侧边栏分类菜单。                                  |
| `licenseName` | **Unlicensed**                                | 版权协议名称，显示在文章底部或“关于”小字。                               |
| `author`      | **emn178**                                    | 作者署名。                                                |
| `sourceLink`  | **<https://github.com/emn178/markdown>**      | 原文/源码链接，方便读者跳转到仓库查看源文件。                              |
| `draft`       | **false**                                     | 不是草稿，允许被编译和发布；设为 `true` 时整篇会被忽略。                     |
| `date`        | **2025-01-20**                                | 部分主题用 `date` 做排序或显示，优先级可能高于 `published`。             |
| `image`       | **{ url, alt }**                              | 封面图：给出完整 URL 和替代文本，用于社交媒体卡片、列表页缩略图。                  |
| `pubDate`     | **2025-01-20**                                | RSS、JSON Feed 里常用的“发布日期”字段，与 `published` 重复时以主题模板为准。 |
### 这个是完整版，自己需要自己改
```markdown
---
title: Markdown Tutorial
published: 2025-01-20
pinned: true
description: A simple example of a Markdown blog post.
tags: [Markdown, Blogging]
category: Examples
licenseName: "Unlicensed"
author: emn178
sourceLink: "https://github.com/emn178/markdown"
draft: false
date: 2025-01-20
image:
  url: 'https://example.com/image.jpg'
  alt: '图片描述'
pubDate: 2025-01-20
---
```
